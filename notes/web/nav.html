Introduction

网站技术对于技术工作者是非常重要的，它不仅是展示自己技术水平的平台，还有利于知识的交流。

由于网站制作已经成为一个比较成熟的领域。故在本章节中，笔者将分享 入门策略，并提供一些优秀的课程链接来辅助学习。

祝各位学习快乐🤗

How to learn?

作为萌新小白，我们并不需要过于复杂的网站设计，所以这里我强烈推荐你们制作静态网站。 其涉及到的技术有 html、CSS 和 JaveScripts。基本只需要这三板斧就可以满足制作静态网站的所有条件。

  1. 学习简单的 html、css 和 javascripts 的内容； 3小时前端入门教程（HTML+CSS+JS）

  2. 开始实战

对于网站制作、Python等工具的学习，我个人推荐的策略是迅速了解其基本框架（比如一个html文件由哪些部分组成）。 之后就是进行多次实战。我也建议读者在学习完其基本框架后，尝试制作自己的个人网站。

Useful webs

快速学习完基本框架，刚开始实战时可能会经常卡壳，我们可以充分利用AI工具帮助我们编写。同时我也推荐一些网站供查询：

  • MDN Web Docs

    该网站提供HTML和CSS等框架的资料，可以通过查询该文档，了解某些类的特定属性等；

  • Shields

    该网站提供badges的制作，你可以根据上面的提示，制作自己的Github badges；

  • Bootstrap

    该网站提供一整套HTML、CSS 和 JavaScript 组件，便于更快速、一致地构建美观、响应式的网站；

Structure

🧩 Layout

作为初学者，我们对于文本结构的布局并不熟悉，这里我提供一个简单且常见的框架： 


<body>
<header>网站头部（Logo, 导航栏）</header>
<nav>导航菜单（可选）</nav>
<main>
  <!-- 页面主体内容 -->
  <section>一个独立的内容块</section>
  <section>另一个内容块</section>
</main>
<footer>页面底部（版权声明、社交链接）</footer>
</body>
